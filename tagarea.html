Features

A very simple tag-input plugin for jQuery. When your users are tagging,
This offers a simple but necessary and sufficient tool to them.
  • Add a new tag by inputing a keyword in the search box
  • Filter tag candidates in the taglist each time a keyword is input
  • Add a tag by clicking/tapping a candidate tag in the taglist
  • Select a candidate tag with an up and down key
  • Add the selected tag with an enter key
  • Implement multiple tag-input interfaces in one page

Minimal Setup

<script src="//ajax.googleapis.com/ajax/libs/jquery/1.8.3/jquery.min.js"></script>
<script src="./tagarea.js"></script>
<script type="text/javascript">
	$(function(){
		$('#country').tagarea({
			taglist_url: './country.php'
		});

		$('#animal').tagarea();
	});
</script>

<div id="country" class="tagareabox">
	<div class="pre-tag">["United States", "United Kingdom", "Japan"]</div>
</div>

<div id="animal" class="tagareabox">
	<div class="pre-tag">["elefant"]</div>
	<div class="pre-list">["panda","giraff","elefant", ... ,"zebra"]</div>
</div>
